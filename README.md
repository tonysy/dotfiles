# dotfiles
Collection of configuration used on Mac or Linux(dot files)

## VIM Configuration

```shell
mv light_vimrc ~/.vimrc

# in your anaconda env 
conda install vim 

# excute vim to install the plugins
vim 
```

## Tmux Configuration

```shell
mv tmux.conf ~/.tmux.conf
```
